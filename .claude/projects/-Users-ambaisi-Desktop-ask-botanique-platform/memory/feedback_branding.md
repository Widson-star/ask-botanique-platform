Stick to the confirmed Botanique Designers brand palette. Do not introduce new color palettes or fonts unilaterally.

**Palette (confirmed from Botanique Designers brand documents):**
- Primary dark teal: `#1a5d5d` — headings, dark text, anchor color
- Accent light teal: `#3E9B9B` — CTAs, interactive elements, badges, hover states
- Hover/darker accent: `#2E8080`
- Deep dark (CTA section bg / footer): `#0d3b3b`
- Surface teal tint (light bg): `#F0F9F9`
- Border: `#C8E8E8`
- Icon bg / badge bg: `#E4F4F4`
- Muted text on dark: `#7DCECE`

**Typography:**
- Font: Quicksand only (400, 500, 600, 700, 800 weights). DM Serif Display was removed.
- Headings: Quicksand 800, letter-spacing -0.025em to -0.03em
- Body: Quicksand 400–600

**Why:** The previous emerald green (#1D9E75 / #0F3D2E) did not match the actual Botanique Designers brand. The real brand uses a dark teal (#1a5d5d) confirmed from expense reports, estimates, and letterhead documents.

**How to apply:** Always use these exact hex values for any new UI work. Never introduce a green-only or forest-green palette — the brand reads as teal, not green.
